# Online-health-forum

This project allows you to check your various symptoms with the various javascript arrays we have and use django channels and jquery, we used redis for client-server interactions 

src folder contains contains all the django files, scripts, templates, static folders and packages 

To run the project
install python 
pip install -r requirements.txt 
download redis 
python manage.py runserver
